- **Account Information**: Email address, username, and profile information when you create an account
- **User Content**: Drawings, sketches, and creative content you create within the App
- **AI Prompts**: Text descriptions and prompts you provide for AI image generation
- **Communication Data**: Information when you contact our support team
- **Payment Information**: Purchase history and transaction IDs (payment processing handled by Apple)

### Information Collected Automatically

- **Device Information**:
  - Device model and operating system version
  - Unique device identifiers (for analytics and crash reporting)
  - Screen resolution and device settings
  - Language and time zone settings

- **Usage Data**:
  - App features used and interaction patterns
  - Session duration and frequency
  - Error logs and crash reports
  - Performance metrics

- **Analytics Data**: We use analytics tools to improve App functionality, which may collect:
  - App launch and feature usage statistics
  - In-app navigation patterns
  - Feature adoption rates

### iOS Permissions

Our App may request the following iOS permissions:

- **Photo Library**: To save your drawings and access images for editing (optional)
- **Camera**: To capture photos for drawing reference (optional)
- **Notifications**: To send you updates about your creations (optional)
- **Network Access**: Required for AI processing and cloud features

You can manage these permissions at any time through your iOS Settings.
